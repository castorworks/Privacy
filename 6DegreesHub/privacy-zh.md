# 隐私政策

生效日期：2026年3月18日

本隐私政策适用于 6DegreesHub（App 名称：`6DegreesHub`，以下简称“本应用”）。

本应用是一款用于创建、编辑、探索和管理知识图谱/关系网络的工具类应用。我们重视您的隐私，并尽量采用本地优先的方式处理数据。请您在使用前仔细阅读本政策。

## 1. 我们处理哪些信息

### 1.1 您在本地创建和保存的内容

当您使用本应用时，您可以在设备上创建、编辑和保存以下内容：

- 空间、节点、关系、自定义类型
- 节点名称、备注、关系标签、图谱结构
- 您为节点手动选择的图片
- 应用设置，例如显示偏好、语言、应用锁开关
- 本地 AI 使用统计信息，例如请求次数、token 用量、延迟和失败类型

上述信息默认保存在您的设备本地存储中，用于实现图谱创建、编辑、搜索、导入导出、分享和设置管理等核心功能。

### 1.2 iCloud 同步数据

如果您已在 Apple 设备上登录 iCloud，且本应用可用 CloudKit 同步功能，本应用会通过 Apple 提供的 iCloud/CloudKit 私有数据库在您的设备之间同步以下数据：

- 空间信息
- 节点、关系、自定义类型及相关文本内容
- 节点图片
- 与图谱相关的结构化数据

这些数据用于实现跨设备同步，不用于广告，也不用于跨 App 跟踪。

### 1.3 AI 功能相关数据

本应用支持由用户自行配置的第三方大语言模型（LLM）服务。只有当您主动配置并使用 AI 功能时，相关数据才会被发送到您所选择的第三方 AI 服务提供方。

可能发送的内容包括：

- 当前输入的文本请求
- 当前图谱的相关上下文信息，例如空间名称、节点名称、关系标签、备注、自定义类型
- 语音命令转写文本
- 为生成导入草稿、图谱问答、图谱修复建议、语音纠错等功能所需的图谱快照或片段

这些数据仅用于完成您主动发起的 AI 请求。本应用不会因为您仅安装本应用而自动向第三方 AI 服务发送图谱内容。

请注意：第三方 AI 服务提供方可能根据其自身隐私政策处理您提交的数据。若您的图谱中包含敏感个人信息，请谨慎使用 AI 功能。

### 1.4 麦克风与语音识别

当您主动使用语音命令功能时，本应用会请求：

- 麦克风权限
- 语音识别权限

这些权限仅用于录制并转写您的语音图谱指令。语音识别能力由 Apple 系统服务提供，相关系统级处理受 Apple 自身条款和隐私政策约束。

本应用不会将您的原始录音作为广告用途处理，也不会将原始录音上传到我们自建服务器。语音转写文本仅会在您实际使用 AI 语音相关能力时，按照本政策“AI 功能相关数据”章节所述，发送给您配置的第三方 AI 服务。

### 1.5 Face ID / Touch ID 应用锁

如果您启用应用锁功能，本应用会调用 Apple 的生物识别认证能力（例如 Face ID 或 Touch ID）来保护本地数据访问。

- 本应用不会读取、保存或上传您的面容数据或指纹数据
- 生物识别数据由 Apple 在设备层面处理
- 本应用仅接收认证是否成功这一结果

### 1.6 内购与购买恢复

本应用使用 Apple StoreKit 提供应用内购买、恢复购买和权限校验功能。

- 支付信息由 Apple 处理
- 本应用不会保存您的银行卡等支付信息
- 本应用会在本地根据 Apple 返回的购买状态判断您是否已解锁专业版功能

### 1.7 API Key

如果您配置 AI 服务 API Key，本应用会将该 Key 保存在设备 Keychain 中，用于后续向您指定的 AI 服务发起请求。

- API Key 不会被上传到我们的自建服务器
- 您可以在应用设置中修改或清除相关配置

## 2. 我们如何使用这些信息

我们处理上述信息的目的仅限于以下范围：

- 提供图谱创建、编辑、搜索、问答、导入导出和分享功能
- 在您的设备之间同步图谱数据
- 响应您主动发起的 AI 请求
- 保护应用访问安全
- 判断专业版购买状态并提供对应功能
- 维护必要的本地设置和本地使用统计

## 3. 我们如何共享信息

我们不会将您的个人信息出售给广告商，也不会将数据用于跨 App 跟踪。

在以下情形下，您的数据可能会被共享或传输给相应服务提供方：

- Apple iCloud / CloudKit：用于跨设备同步您选择保存的图谱数据
- Apple StoreKit：用于处理购买、恢复购买和权限校验
- Apple 语音识别相关系统服务：用于您主动发起的语音转写
- 您自行配置的第三方 AI 服务提供方：用于完成您主动发起的 AI 请求

除上述情形外，我们不会因为本应用的正常使用而将您的图谱内容上传到我们的自建服务器。

## 4. 数据保留

我们通常按以下方式保留数据：

- 本地数据：在您设备上保留，直到您删除相关内容、重置应用数据或卸载应用
- iCloud 同步数据：在您的 iCloud 私有存储中保留，直到您删除相关内容或关闭相关同步环境
- AI 使用统计：保留在设备本地，直到您在应用内手动重置
- API Key：保留在设备 Keychain 中，直到您主动修改、清除，或系统删除相关 Keychain 项

对于发送到第三方 AI 服务的数据，其保留期限由相应第三方服务提供方决定，我们无法直接控制。

## 5. 您的选择与控制

您可以通过以下方式控制您的数据：

- 不使用 AI 功能，或不配置 AI 服务
- 删除节点、关系、空间或节点图片
- 使用应用内导出功能导出您自己的数据
- 在 iOS 设置中管理麦克风、语音识别和 iCloud 权限
- 在应用设置中启用或关闭 Face ID / Touch ID 应用锁
- 在应用设置中修改或清除 AI 服务配置

如果您所在地区的法律赋予您访问、更正、删除或限制处理个人信息的权利，您也可以通过 App Store 页面中的支持联系方式与我们联系。

## 6. 儿童隐私

本应用并非专门面向儿童设计。我们不会故意通过本应用向儿童定向收集个人信息。

## 7. 国际传输

如果您使用第三方 AI 服务，相关数据可能会被传输到您所选服务提供方所在的国家或地区并在当地处理。不同服务提供方适用的隐私规则可能不同，请您在使用前自行查阅相关政策。

## 8. 隐私政策更新

我们可能根据功能变化、法律要求或产品调整更新本隐私政策。若发生重大变化，我们会通过更新应用内链接、支持页面或应用版本信息的方式发布更新版本。更新后的政策自公布之日起生效。

## 9. 联系我们

如您对本隐私政策或本应用的数据处理方式有疑问，请通过 App Store 页面中的支持链接或开发者公开支持页面与我们联系。
